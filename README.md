# HR Analytics: Salary Intelligence System 📊

## Overview
This project analyzes employee data to uncover insights related to salary, performance, and attrition.

## Objectives
- Analyze employee attrition
- Compare salary across departments
- Identify trends in performance and experience
- Build interactive dashboards for decision-making

## Tech Stack
- SQL (MySQL)
- Python (Pandas, NumPy)
- Power BI
- Excel

## Features
- Attrition rate calculation
- Department-wise salary analysis
- KPI dashboard (Power BI)
- Data cleaning and EDA

## Dataset
Synthetic HR dataset with employee details such as salary, department, experience, and attrition.

## How to Run
1. Import dataset into MySQL
2. Run SQL queries
3. Load data into Power BI
4. Build dashboards

## Future Improvements
- Add machine learning for attrition prediction
- Real-time data integration
- Advanced KPI tracking
